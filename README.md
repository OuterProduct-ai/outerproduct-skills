# OuterProduct MCP Skills

A collection of skills built for the OuterProduct MCP server. Each skill packages a domain-specific workflow — guiding a non-technical user from raw data through model training, analysis, and a polished deliverable — on top of OuterProduct's explainable AI tooling.

## Skills

### [filing-suspicious-activity-reports](filing-suspicious-activity-reports/)

For fraud and BSA/AML analysts investigating a specific subject. The skill trains a detector from labeled case data, evaluates the subject against learned fraud patterns, and — on request — drafts the SAR narrative needed to support a regulatory filing. Output is framed in investigator language with every claim traceable to a specific behavior.

### [building-churn-winback-briefings](building-churn-winback-briefings/)

For Customer Success and retention managers preparing for an executive review. The skill trains a churn model from customer data, surfaces the top drivers of attrition, segments the churners into distinct stories, and recommends a priority segment with an outreach approach. The final deliverable is a styled HTML briefing a VP can skim in 90 seconds, including one specific customer to call before launching the campaign.
